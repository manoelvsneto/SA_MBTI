# SA_MBTI: MBTI Personality Profiles in Software Architecture Education

## About
Research investigating whether MBTI personality types influence academic performance 
in Software Architecture courses. A study analyzing 102 students from PUCPR's Software 
Architecture discipline.

## Research Question
Does personality (MBTI) matter in Software Architecture education and performance?

## Dataset
- **Samples**: 102 students
- **Columns**: Class, MBTI Type, Year, Gender, Architecture Requirements Grades, 
  System Development Grades, Objective Exam, Final Grade, MBTI Group, Profile Name
- **MBTI Groups**: Analysts, Diplomats, Explorers, Sentinels

## Methodology
- Personality assessment via 16Personalities (MBTI-inspired)
- Performance evaluation: Team exercises (40%) + objective exam (60%)
- Statistical analysis: Kruskal-Wallis tests with epsilon-squared effect sizes

## Files
- `SBES_MTBI_SA.ipynb` - Analysis script (Google Colab compatible)
- `dados.xlsx` - Raw dataset
- `data.md` - Formatted dataset table
- `main.tex` - Academic paper (SBES 2026 submission)
